Tutorials

Master Alight Motion with step-by-step guides.

Install

How to install presets

Import preset ZIP files into Alight Motion on Android and iOS.

01Download the ZIP from your dashboard
02Extract to get the .aep or .preset file
03Open Alight Motion → Projects → Import
04Select the file and tap Import
05Find it in your presets library
Create

How to create presets

Build custom presets from scratch and export as ZIP.

01Create a new project in Alight Motion
02Add effects, keyframes, and animations
03Select all layers → Long press → Save as Preset
04Name your preset and add a thumbnail
05Export as ZIP to share
Effects

How to use effects

Master blur, glow, color grading, and transitions.

BlurAdd → Effects → Blur. Keyframe intensity for smooth transitions
GlowEffects → Glow. Adjust radius and intensity
ColorEffects → Color Correction → Curves
CutUse opacity + scale keyframes for smooth transitions
Curve Editor

Curve / graph editor

Master easing and smooth motion for professional animations.

InEase In — slow start, fast end. Drag first handle down
OutEase Out — fast start, slow end. Drag last handle up
SS-curve for natural motion — both handles curved
TipCopy curve values between properties for consistency
Advanced

Beat sync editing

Sync cuts and effects to music beats for viral content.

01Import audio track to the timeline
02Enable waveform view to see beat markers
03Place keyframes exactly on beat peaks
04Use scale/rotation snaps on each beat
05Add flash/glow effects on the downbeats
Advanced

Velocity editing

Create smooth speed ramps and velocity edits.

RampUse time remapping to slow down or speed up clips
BlurAdd radial blur during fast sections
MotionEnable motion blur for realistic movement
CutUse scale + opacity for smooth velocity cuts
